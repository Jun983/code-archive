# Java

Java는 JVM 위에서 실행되는 정적 타입 객체지향 언어로, "Write Once, Run Anywhere"를 지향합니다.

이 문서에서는 기본 문법·클래스·상속·인터페이스 등 핵심 개념부터 제네릭, 컬렉션, 람다·스트림 API, JVM 내부 구조(메모리, GC)와 멀티스레드 동시성 제어를 정리합니다.
